# PolyRenderer
PolyRenderer is an easy-to-use Rendering engine that Renders instances in a low-FPS style

## Example

An example code on how to use PolyRenderer:

```lua
local PolyRenderer = require("path/to/module")
local PartOrModel = "your part or model"

PolyRenderer.Add(PartOrModel)
```
